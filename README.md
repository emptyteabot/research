# LeadPulse Assignment Site

这是一个为 `AI 驱动端到端开发` 面试作业准备的静态展示站点。

## 站点内容

- `index.html`：作业单页结构
- `styles.css`：浅色展示风格与弹窗、卡片、MVP 视觉
- `main.js`：竞品调研交互与 LeadPulse MVP 拆解逻辑
